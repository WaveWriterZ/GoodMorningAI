Contributing to GoodMorningAI

Thank you for your interest in contributing to GoodMorningAI and the Mission Systems ecosystem.

Our Mission

We are building an open, living knowledge system that helps people and organizations preserve knowledge, strengthen learning, improve governance, and collaborate responsibly across generations.

Our Principles

Every contribution should strive to:

- Preserve historical context.
- Distinguish evidence from interpretation.
- Welcome respectful critique.
- Improve clarity.
- Document decisions.
- Leave the project better than it was found.

Contribution Process

1. Open an Issue describing the problem or proposal.
2. Discuss the idea with the community.
3. Create a feature branch.
4. Submit a Pull Request.
5. Participate in review and revision.

Research Contributions

Research submissions should clearly distinguish:

- Documented observations
- Existing scholarship
- Mission Systems synthesis
- Working hypotheses
- Supporting evidence
- Alternative explanations
- Open questions

Code Contributions

Code should be:

- Well documented
- Tested where practical
- Modular
- Readable
- Maintainable

Community Expectations

We value:

- Curiosity
- Integrity
- Humility
- Transparency
- Stewardship
- Continuous learning

Thank you for helping build GoodMorningAI.
